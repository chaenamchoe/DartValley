사용설명서

다트피쉬 프로그램에 데이터발리의 태깅을 변환해서 임포트하도록 해주는 프로그램
로컬에 데이터베이스를 사용하고 서버에서는 사용자 승인처리만 하는 방식임.
서버: 210.122.7.39 d:\fb_data\DartVolley\dart_volley.fdb 에 
사용자 승인 테이블 login_user 에서 승인을 해줘야 사용이 가능함.

1. 설치방법
- 설치파일을 이용해서 설치
- 로컬 데이터 파일이 포함되어야 함.(DATAVOLLY.FDB)

2. 사용자 승인
- 사용자 승인테이블에서 직접 승인처리 approved = 1로 바꾼다.
- 사용자가 처음 등록을 할 때 컴퓨터의 맥어드레스를 사용자관리 테이블에 같이 저장한다.
- 로그인 과정에서 사용자PC의 Mac address와 사용자ID로 사용자테이블에 등록여부를 확인한다.
- 사용자ID와 PC Mac address가 같아야만 사용이 가능하다.


3. 데이터 임포트를 해서 사용
- 데이터발리 .dvw 파일을 임포트 하면 모든 데이터를 테이블에 저장한다.
- 엑셀로저장을 통해 csv 파일을 생성해 준다.
- 해당 csv 파일을 다트피쉬 프로그램에서 임포트 하여 사용.
